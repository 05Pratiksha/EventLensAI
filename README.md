# EventLens AI

A premium, AI-powered event photo discovery and delivery platform frontend built with Next.js, Tailwind CSS, and Framer Motion. 

## 🚀 Features

- **Dynamic Dark-Mode First Design:** Uses curated color palettes, sleek dark modes, and modern aesthetics.
- **Micro-Animations & Smooth Transitions:** Built with Framer Motion for enhanced user experience.
- **Responsive Studio Dashboard:** Beautifully crafted layouts for studio owners.
- **Seamless Media Upload Workflows:** Streamlined and intuitive media management.
- **3D Elements:** Uses Three.js for interactive landing experiences.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **3D/Graphics:** [Three.js](https://threejs.org/), [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/), [React Query](https://tanstack.com/query/latest)

## 🏃‍♂️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view it in your browser.
